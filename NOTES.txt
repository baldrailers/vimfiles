*tpope/vim-rails*
Enables |gf| for files.
See: http://ricostacruz.com/cheatsheets/vim-rails.html

Commands: ~

     |:A|   alternate file
     |:R|   related file

Abbreviations: ~

    AC::   ActionController
    AR::   ActiveRecord
    ...

Extracting partials: ~

    `:Rextract` `home/foo_partial`

App: ~

    :Econtroller <file>  # app/controllers/*_controller.rb
    :Ehelper             # app/helpers/*_helper.rb
    :Emodel <file>       # app/models/*.rb
